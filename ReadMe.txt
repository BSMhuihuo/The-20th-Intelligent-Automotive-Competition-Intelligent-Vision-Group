运行All_know.py：
    recognize_more()  验证所有的数据集，并计算指标，其中数字数据集没有用与工具集相似路径的数据集，因为在生成判别器数据时用了数据增强，
                      但为了保证训练过程中所使用的训练集与测试阶段使用的增强图像数据源彼此独立，展现鲁棒性，又为了测试单独生成了增强数据
    recognize() 验证单个图像

以下是所有文件说明：
    struct: 判别器相关文件（判别是数字还是工具）
    num:识别数字的模型相关文件
    tool:识别工具的相关文件
    _model:用来保存模型
    _train:用来训练
    _know:用来验证模型
特别的：
    num_know_onpic:用来直接识别图片上的数字并且框选出来然后识别标注到图像中
    plot.py:用来生成t-SNE和CAM的图片
    gen_digist_num.py:用来生成电子数字手写图像
    gen_pic_num.py:用来生成纸质手写数字图像
    struct_gendata:为判别器准备数据，将所有的数字图片和工具图片混在一起训练，用来判别数字还是工具
